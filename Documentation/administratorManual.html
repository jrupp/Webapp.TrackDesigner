Create own shapes

  1. Create a picture of your shape, store it to "Resources/Img/Tracks/"
  2. Add description of your shape to "Configuration/ShapeConfiguration.ts": 
    
longCurve: new ShapeType(
	"longCurve",
	"Long curve",
	[new Vector(62.3424, 0.9906, 0.96875),new Vector(62.3424, 0.5094, 0.53125)],
	[
		new Variant(ShapeConfiguration.variantTypes['Default'], 255.2, 255.2, "Resources/Img/Tracks/longCurved-Default.png")
	]
)

    tsc Main.ts --module AMD
